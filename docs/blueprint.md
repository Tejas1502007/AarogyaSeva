# **App Name**: SECURE-SEVA

## Core Features:

- Role-Based Access: Secure patient login with distinct roles: User, Doctor, Pharmacist, Admin.
- Secure Upload: Allows patients to upload health records (PDFs, DICOM, lab results) with client-side encryption.
- Secure Sharing: Generates time-limited QR codes/tokens for secure record sharing.
- Record Display: Clear display of health records with easy navigation and search functionalities.
- AI Record Summarization: AI-powered summarization tool that allows medical professionals to quickly and concisely overview records.

## Style Guidelines:

- Primary color: Teal (#008080) for trustworthiness and health.
- Background color: Light teal (#E0F8F8) for a calm, clean interface.
- Accent color: Soft lavender (#E6E6FA) to add a touch of elegance.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Simple, professional icons for document types and functions.
- Clean, intuitive layout with a focus on readability and easy navigation.
- Subtle transitions and animations for a smooth user experience.
